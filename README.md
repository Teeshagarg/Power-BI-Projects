# Sentiment Analysis of Trump vs Clinton Facebook Posts using Power BI

## 📌 Project Overview

This project performs sentiment analysis on Facebook posts related to Donald Trump and Hillary Clinton during the U.S. Presidential Election campaign. The aim is to analyze public opinion, compare sentiment patterns between the two candidates, and present insights through an interactive Power BI dashboard.

Using sentiment analysis and business intelligence techniques, the project transforms social media data into meaningful visualizations that help understand public perception and engagement.

---

## 🎯 Objectives

* Analyze sentiments expressed in Facebook posts related to Donald Trump and Hillary Clinton.
* Classify posts into Positive, Negative, and Neutral categories.
* Compare overall public sentiment towards both candidates.
* Create interactive Power BI dashboards for visual exploration.
* Identify trends and patterns in social media discussions.

---

## 🛠 Tools & Technologies

* Power BI
* Microsoft Excel / CSV

* Data Cleaning and Transformation Techniques
